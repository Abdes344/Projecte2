# 🌐 T09: Donar-se d'Alta en un Hosting Gratuït

## 📖 Breu descripció

En aquesta activitat s'ha realitzat una investigació sobre diferents proveïdors de **hosting gratuït** amb l'objectiu de disposar d'un entorn real per fer pràctiques de desenvolupament i administració web.

Després d'analitzar diverses alternatives, s'ha seleccionat un proveïdor que permet allotjar un lloc web i instal·lar **WordPress** de manera gratuïta.

---

# 🎯 Objectius

* Investigar diferents serveis de hosting gratuït.
* Comparar les característiques dels proveïdors.
* Crear un compte al servei seleccionat.
* Activar l'espai de hosting.
* Instal·lar WordPress.
* Verificar el correcte funcionament del lloc web.

---

# 🔍 Investigació de Proveïdors

Durant la investigació s'han analitzat diverses opcions de hosting gratuït.

# 🏆 Proveïdor Seleccionat

## InfinityFree

### Motius de l'elecció

* ✔️ Totalment gratuït.
* ✔️ Sense límit de temps.
* ✔️ Compatible amb WordPress.
* ✔️ Inclou PHP i MySQL.
* ✔️ Certificat SSL disponible.
* ✔️ Fàcil configuració.
