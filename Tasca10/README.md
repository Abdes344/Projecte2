# 🧠 T10: Màsterclass de Markdown

## 📖 Breu descripció

En aquesta activitat s'ha realitzat una **màsterclass de Markdown**, un llenguatge de marcatge lleuger molt utilitzat en entorns professionals com **GitHub, GitLab i documentació tècnica de projectes**.

Markdown permet crear documents estructurats de manera senzilla, facilitant la lectura, l'organització i la mantenibilitat de la documentació en projectes de desenvolupament i sistemes.

---

# 🎯 Objectius específics

* Comprendre què és Markdown i la seva utilitat professional.
* Aprendre a estructurar documents amb títols i subtítols.
* Crear llistes ordenades i no ordenades.
* Elaborar taules comparatives.
* Utilitzar checklists per controlar tasques.
* Inserir enllaços i imatges en documents tècnics.
* Integrar Markdown amb GitHub en un flux de treball real.

---

# 🧱 Estructura d'un document Markdown

Markdown permet organitzar la informació de manera jeràrquica:

## 🟢 Títols

```markdown id="t1hd"
# Títol principal
## Subtítol
### Subsubtítol
```

---

## 📋 Llistes

### Llista no ordenada

* Element 1
* Element 2
* Element 3

### Llista ordenada

1. Primer pas
2. Segon pas
3. Tercer pas

# 🔗 Integració amb GitHub

Markdown és el format estàndard utilitzat a:

* GitHub README.md
* Issues i Pull Requests
* Wikis de projectes
* Documentació tècnica

Això permet treballar en entorns col·laboratius de manera eficient i professional.
